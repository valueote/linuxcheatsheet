---
title: cut
description: 从每行文本中按列/字符/字节截取内容。
tags: [文本, 处理]
flags:
  - { name: "-d", description: "指定分隔符（默认制表符）" }
  - { name: "-f", description: "选择字段（列）号，如 1,3 或 1-3" }
  - { name: "-c", description: "按字符位置截取，如 1-10" }
  - { name: "-b", description: "按字节位置截取（多字节字符需注意）" }
examples:
  - { title: "按逗号分隔取第 1 列", command: "cut -d, -f1 data.csv" }
  - { title: "取第 1-3 列", command: "cut -d, -f1-3 data.csv" }
  - { title: "取每行前 10 个字符", command: "cut -c1-10 file.txt" }
---

用途：按列或字符位置裁剪文本。

语法：

```bash
cut [选项] [文件...]
```

常见用法：

```bash
# CSV 取第 1 列
cut -d, -f1 data.csv

# 取第 1-3 列
cut -d, -f1-3 data.csv

# 取前 10 个字符
cut -c1-10 file.txt
```
