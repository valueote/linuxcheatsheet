用途：为文件/目录创建引用。软链接类似 Windows 的快捷方式，硬链接为同一 inode 的另一名称。

语法：

```bash
ln [选项] 源 目标
```

常见用法：

```bash
# 将可执行加入 PATH（软链接）
ln -s /opt/app/bin/app /usr/local/bin/app

# 创建硬链接（同盘）
ln file.txt file.hardlink

# 覆盖已存在的符号链接
ln -sf new-target current-link
```
