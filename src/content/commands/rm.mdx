用途：删除文件或目录，删除操作不可逆，务必谨慎。

语法：

```bash
rm [选项] 路径...
```

<Danger>
强烈建议先用 `ls` 或 `find -print` 预览目标，再执行删除。
避免直接使用 `-f`；涉及目录时慎用 `-r`。
</Danger>

常见用法：

```bash
# 删除前进行确认
rm -i file.txt

# 删除空目录（也可用 rmdir）
rm -d empty_dir

# 递归删除目录（危险，先确认路径再执行）
rm -r ./tmp
```
