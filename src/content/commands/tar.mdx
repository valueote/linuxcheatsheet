用途：将多个文件/目录打包为一个归档文件，或从归档中解包。

语法：

```bash
tar -c|x|t [选项] -f 文件.tar[.gz|.bz2|.xz] [路径...]
```

常见用法：

```bash
# 创建 gzip 压缩的归档
tar -czf archive.tar.gz dir/

# 解压
tar -xzf archive.tar.gz

# 查看归档内容
tar -tzf archive.tar.gz

# 解压到指定目录
tar -xzf archive.tar.gz -C /tmp/output
```

<Danger title="解压前先查看">
解压到当前目录可能覆盖同名文件。建议先用 `-t` 列出内容，或解压到独立目录（`-C <dir>`）。
</Danger>
