用途：在文件或标准输入中查找匹配模式的行。

语法：

```bash
grep [选项] 模式 [文件...]
```

常见用法：

```bash
# 递归搜索并显示行号/文件名
grep -rinH "TODO" .

# 反向匹配（排除包含关键字的行）
grep -v "test" file.txt

# 扩展正则：匹配多个关键词
grep -E '(error|fail|fatal)' app.log
```
