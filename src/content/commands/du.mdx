  - { title: "查看各子目录大小（简洁）", command: "du -sh *" }

用途：评估目录层级的磁盘占用，用于发现“大户”。

语法：

```bash
du [选项] [路径]
```

常见用法：

```bash
# 每级目录大小（一层）
du -h --max-depth=1

# 当前目录总大小
du -sh .

# 列出各子目录大小
du -sh *
```
