用途：浏览文件尾部与实时追加内容（如日志）。

语法：

```bash
tail [选项] [文件]
```

常见用法：

```bash
# 查看最后 100 行
tail -n 100 app.log

# 实时跟随日志（文件轮转时用 -F 更稳）
tail -f app.log

# 跟随并过滤出 ERROR
tail -f app.log | grep ERROR
```
