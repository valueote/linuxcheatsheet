---
title: df
description: 显示文件系统磁盘空间占用。
tags: [系统管理, 磁盘管理]
flags:
  - { name: "-h", description: "人类可读的单位（KB/MB/GB）" }
  - { name: "-T", description: "显示文件系统类型" }
  - { name: "-i", description: "显示 inode 使用情况" }
  - { name: "-x", description: "排除某类文件系统" }
examples:
  - { title: "查看磁盘使用（人类可读）", command: "df -h" }
  - { title: "包含文件系统类型", command: "df -hT" }
  - { title: "查看 inode 使用", command: "df -hi" }
---

用途：查看各挂载点的容量与使用率。

语法：

```bash
df [选项] [路径]
```

常见用法：

```bash
# 总览
df -h

# 带类型
df -hT

# inode 使用情况
df -hi
```
