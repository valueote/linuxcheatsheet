用途：复制文件到新位置，或复制整个目录。

语法：

```bash
cp [选项] 源 目标
cp [选项] 源... 目录/
```

常见用法：

```bash
# 复制文件
cp src.txt dest.txt

# 递归复制目录
cp -r src_dir dest_dir/

# 保留权限/时间等属性
cp -a src_dir dest_dir/

# 覆盖前确认（更安全）
cp -i src.txt dest.txt

# 不覆盖已存在的文件
cp -n src.txt dest.txt
```

import Danger from '../../components/Danger.astro'

<Danger title="注意覆盖">
复制时若不加 `-i` 或 `-n`，将直接覆盖目标文件。建议默认加 `-i` 或在操作前先备份。
</Danger>
