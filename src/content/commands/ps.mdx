用途：列出系统进程，可配合 `grep` 过滤或按列排序。

语法：

```bash
ps [选项]
```

常见用法：

```bash
# 所有进程
ps aux

# 自定义列并按 CPU 排序
ps -eo pid,ppid,%cpu,%mem,cmd --sort=-%cpu | head
```
