用途：查看当前或指定目录的文件与子目录。

语法：

```bash
ls [选项] [路径]
```

常用选项：

```text
-l   长列表格式
-a   显示隐藏文件
-h   人类可读大小（与 -l 一起）
-S   按大小排序
-t   按时间排序
```

常见用法：

```bash
# 长列表 + 隐藏 + 人类可读大小
ls -alh

# 按修改时间降序
ls -lt

# 只显示目录
ls -d */
```
