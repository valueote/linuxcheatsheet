用途：设置访问权限。可使用符号法（u/g/o/a + r/w/x +/-）或数字法（如 644, 755）。

语法：

```bash
chmod [选项] 模式 文件...
```

常见用法：

```bash
# 符号法：给所有者添加执行
chmod u+x script.sh

# 数字法：rw-r--r--
chmod 644 file.txt

# 批量：目录 755，文件 644
find . -type d -exec chmod 755 {} +
find . -type f -exec chmod 644 {} +
```
