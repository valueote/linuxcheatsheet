用途：在不中断管道的情况下把输出“分叉”保存到文件。

语法：

```bash
... | tee [选项] 文件...
```

常见用法：

```bash
# 保存构建输出到文件，同时在终端可见
make build 2>&1 | tee build.log

# 追加记录运行日志
python script.py | tee -a run.log
```
