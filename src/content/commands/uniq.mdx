用途：对相邻的重复行去重；通常需要先对输入排序。

语法：

```bash
uniq [选项] [文件]
```

常见用法：

```bash
# 去重（先排序）
sort words.txt | uniq

# 统计每行出现次数并按次数降序
sort words.txt | uniq -c | sort -nr

# 仅显示重复的行
sort words.txt | uniq -d
```
