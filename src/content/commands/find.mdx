用途：在目录树中按名称、大小、时间、类型等条件搜索。

语法：

```bash
find [起始路径] [条件] [动作]
```

常见用法：

```bash
# 名称匹配
find . -type f -name '*.log'

# 大小过滤（>100MB）
find . -type f -size +100M

# 最近一天修改
find . -type f -mtime -1

# 与 xargs 配合，可靠处理空格
find . -type f -name '*.txt' -print0 | xargs -0 wc -l
```
