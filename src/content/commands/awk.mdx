用途：按列读取文本并进行筛选、计算与格式化输出。

语法：

```bash
awk [选项] '模式 { 动作 }' [文件...]
```

常见用法：

```bash
# CSV 第 1 列
awk -F, '{print $1}' data.csv

# 匹配关键字再取列
awk '/error/ {print $2}' app.log

# 聚合求和
awk '{sum+=$3} END {print sum}' data.txt
```
