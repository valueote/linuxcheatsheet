  - { title: "分页查看文件", command: "less -N file.txt" }

用途：更强大的分页器，适合浏览较长的文本输出。

语法：

```bash
less [选项] [文件]
```

常见用法：

```bash
# 分页查看并显示行号
less -N file.txt

# 保留颜色并分页
grep --color=always ERROR app.log | less -R

# 进入跟随模式（按 Ctrl+C 退出跟随，q 退出）
less +F app.log
```
